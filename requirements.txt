# UH2S — зависимости.

# С версии «ленивых импортов» сторонние пакеты загружаются только в момент,
# когда соответствующая функция реально вызывается. Поэтому для ТЕСТОВОГО ЗАПУСКА
# (логин + навигация по UI) достаточно установить только блок «ЯДРО».
# Пакеты коннекторов ставьте по мере того, как реально будете обращаться к источникам.
#
# Рекомендуется Python 3.12 (под 3.14 часть пакетов может не иметь готовых wheel
# и потребует сборки: psycopg2-binary, pymssql, llama-cpp-python).
#
# Минимальный тестовый запуск:
#   python -m venv .venv && source .venv/bin/activate
#   pip install --upgrade pip
#   pip install nicegui cryptography bcrypt
#   python front.py
# Полная установка:
#   pip install -r requirements.txt
